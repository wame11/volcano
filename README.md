# Volcanic Activity Website

One-page website about volcanic activity as a natural cause of climate change.

Files:
- `index.html` - the website
- `styles.css` - red/orange volcano styling and animations
- `script.js` - reveal, counter, and hover animations
- `presentation-speech.md` - 3 minute speech split for 5 presenters

Open `index.html` in a browser, or enable GitHub Pages from the repository settings to share it as a public website.
